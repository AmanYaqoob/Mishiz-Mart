# Category circle images (optional)

The homepage "Shop by ritual" circles already show a product photo by default.
To use a dedicated image instead, drop a SQUARE (1:1) photo here named:

    public/cats/skin.jpg
    public/cats/body.jpg
    public/cats/hair.jpg
    public/cats/scent.jpg

They load automatically (centered crop). If a file is missing, the circle falls
back to the matching product photo, then a soft gradient — so it never breaks.
